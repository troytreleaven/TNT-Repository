## Executive Summary

This document establishes the comprehensive execution management system for Dale Carnegie Training's Ontario & Maritimes operations. The system integrates AI agent capabilities with human team management to achieve $300K+/month revenue targets through structured 12-Week Year planning, daily accountability, and continuous optimization.

### Core Philosophy
- **Speed is secondary** - Quality and accuracy are primary
- **Automation handles tracking** - Humans handle coaching
- **Weekly sprints** - 12-week year methodology
- **Daily pulse** - Consistent check-ins and adjustments

## Part 2: 12-Week Year Planning System

### 2.1 Planning Framework

Each 12-week cycle consists of:

| Week Type | Focus | Activities |
|-----------|-------|------------|
| Weeks 1-3 | Launch | Execute new initiatives, quick wins |
| Weeks 4-6 | Momentum | Build pipeline, convert leads |
| Weeks 7-9 | Push | Close deals, maximize revenue |
| Weeks 10-11 | Harvest | Final pushes, celebrations |
| Week 12 | Review | Retrospective, next cycle planning |

### 2.2 Weekly Planning Process

**Monday 9 AM: Pipeline Meeting (1 hour)**
- Review previous week's results
- Assign hot leads
- Set current week goals
- Identify blockers

**Friday 3 PM: Retrospective (30 min)**
- What worked?
- What didn't?
- Adjustments for next week

### 2.3 Individual 12-Week Plans

Each team member has a 12-week plan with:

| Element | Description |
|---------|-------------|
| **12-Week Goal** | Big revenue target |
| **Weekly Milestones** | Breakdown by week |
| **Daily Actions** | Specific tasks |
| **Metrics** | KPIs to track |
| **Blockers** | Current challenges |
| **Support Needed** | From Troy/Kevin/Jarvis |

| Amy | 60% | Operations heavy | 40% |
| Paul | 100% | Retiring Aug | 80% |

### 6.3 Capacity-Based Targets

When capacity is reduced, targets adjust proportionally:

```
Adjusted Target = Base Target × (Available Capacity / Base Capacity)

Example:
Manu: $50K × (80% / 100%) = $40K for this period
```
