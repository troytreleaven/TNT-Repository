## Summary
Article about maximizing OpenClaw's value beyond just using it like ChatGPT.

## Key Hacks

1. **Use different models for different tasks**
   - Use Opus for brain
   - Codex for coding
   - Minimax 2.5 for research
   - Qwen 3.5 for creative writing
   - Benefits: saves money, parallelization, faster & smarter

2. **Host locally, not VPS**
   - Like having employee in office vs across world
   - Enables airdrop workflows
   - Example: video → transcript → 10 languages → chapters → thumbnail

3. **Telegram for quick, Discord for deep**
   - Telegram: fastest messaging
   - Discord: complex multi-channel, subagent workflows

4. **Reverse prompting**
   - Ask bot questions vs telling it what to do
   - Example: "Based on what you know about me, what's next?"
   - Use when bot is idle

5. **Vibe coding with Codex**
   - Tell agent what to build, it uses Codex CLI itself

6. **Build custom Mission Control**
   - Create custom tooling
   - Prompt: "Build me a Mission Control in NextJS..."

7. **Run every task through OpenClaw**
   - Ask: "how can this be done better?"

8. **Start cheap, scale up**
   - Start with old laptop
   - Scale to Mac Mini/Studio as needed

9. **Don't give email access**
   - Prompt injection risk
   - Automations not great

10. **Don't give it X account**
    - X cracking down on bots
    - API usage being restricted

11. **Have fun and experiment**
    - Don't obsess over monetization
    - Enjoy the technology
