# Workshop Delivery Quality Assurance Framework

**Created by:** Quinn, Quality Control Officer  
**Date:** 2026-02-21  
**Purpose:** Standardized quality assurance process for all workshop deliveries

---

## 1. Pre-Workshop Checklist

### 1.1 Materials & Content (Complete 48hrs before)
- [ ] Course slides/content reviewed and finalized
- [ ] All handouts and worksheets prepared
- [ ] Exercise files/test environments tested
- [ ] Backup materials available (offline versions)
- [ ] Technical demonstrations prepped and verified

### 1.2 Logistics (Complete 24hrs before)
- [ ] Venue confirmed (or virtual platform tested)
- [ ] Room setup arranged (seating, A/V, whiteboard)
- [ ] Participant list confirmed and materials sent
- [ ] Registration/check-in process ready
- [ ] Emergency contacts and contingency plans in place

### 1.3 Facilitator Prep (Complete 24hrs before)
- [ ] Facilitator briefed on all content
- [ ] Run-through of timing and transitions
- [ ] Equipment tested (mic, projector, demo tools)
- [ ] Water, notes, and personal items prepared
- [ ] Backup facilitator identified

### 1.4 Communication
- [ ] Reminder sent to participants ( timing: 1 week, 48hrs, 24hrs)
- [ ] Agenda and learning objectives shared
- [ ] Pre-work assignments distributed (if applicable)
- [ ] Virtual link tested with early joiners (virtual only)

## 2. Post-Workshop Follow-Up Process

### 2.1 Immediate (Same Day)
| Task | Owner | Timeline |
|------|-------|----------|
| Collect feedback forms | Facilitator | End of session |
| Note any issues/digital artifacts | Facilitator | Within 1 hr |
| Send thank you + next steps email | Facilitator | Same day |
| Secure venue/close virtual room | Facilitator | End of session |

### 2.2 Short-Term (Within 1 Week)
| Task | Owner | Timeline |
|------|-------|----------|
| Compile feedback data | QC Officer | 48 hrs post |
| Send post-workshop survey | QC Officer | 24-48 hrs post |
| Review attendance records | Admin | 48 hrs post |
| Process any participant requests | Facilitator | 72 hrs post |

### 2.3 Medium-Term (Within 2 Weeks)
| Task | Owner | Timeline |
|------|-------|----------|
| Analyze feedback and metrics | QC Officer | 7 days post |
| Generate quality report | QC Officer | 10 days post |
| Share report with stakeholders | QC Officer | 10 days post |
| Document lessons learned | Facilitator + QC | 14 days post |
| Archive materials for future use | Admin | 14 days post |

## 3. Quality Metrics to Track

### 3.1 Engagement Metrics
| Metric | Target | Measurement Method |
|--------|--------|---------------------|
| Attendance rate | ≥ 85% | Actual vs. registered |
| Completion rate | ≥ 90% | Participants completing full session |
| Participation in exercises | ≥ 80% | Facilitator observation |
| Questions asked | ≥ 3 per session | Facilitator tally |
| Virtual: Chat/raise hand activity | ≥ 10 interactions | Platform logs |

### 3.2 Satisfaction Metrics
| Metric | Target | Measurement Method |
|--------|--------|---------------------|
| Overall satisfaction score | ≥ 4.2/5.0 | Post-survey |
| Net Promoter Score (NPS) | ≥ 40 | Post-survey |
| Content relevance rating | ≥ 4.0/5.0 | Post-survey |
| Facilitator effectiveness | ≥ 4.5/5.0 | Post-survey |
| Would recommend workshop | ≥ 85% | Post-survey |

### 3.3 Learning Metrics
| Metric | Target | Measurement Method |
|--------|--------|---------------------|
| Learning objective achievement | ≥ 80% | Pre/post assessment or quiz |
| Confidence improvement | ≥ 20% lift | Self-reported pre/post |
| Knowledge retention (30-day) | ≥ 70% | Follow-up quiz |
| Application of skills | ≥ 60% | 30-day follow-up survey |

### 3.4 Operational Metrics
| Metric | Target | Measurement Method |
|--------|--------|---------------------|
| Start time adherence | ± 5 min | Session observation |
| Technical issues count | ≤ 2 minor | Facilitator log |
| Materials accuracy | 100% | QC review |
| Facilitator punctuality | 100% on-time | Session log |

## 5. Quality Scorecard Summary

| Category | Weight | Score (1-5) | Weighted Score |
|----------|--------|-------------|----------------|
| Participant Satisfaction | 30% | | |
| Engagement & Participation | 20% | | |
| Learning Outcomes | 25% | | |
| Operational Excellence | 15% | | |
| Facilitator Performance | 10% | | |
| **Overall Quality Score** | 100% | | |

**Score Thresholds:**
- 4.5 - 5.0: Excellent - Share best practices
- 4.0 - 4.4: Good - Minor improvements only
- 3.5 - 3.9: Needs Improvement - Action plan required
- Below 3.5: Critical - Full review before next delivery

---

## 6. Continuous Improvement Actions

| Trigger | Action |
|---------|--------|
| Score drops >0.5 from baseline | Root cause analysis |
| NPS < 30 | Facilitator coaching |
| Technical issues >3 | Tech audit |
| Low engagement observed | Activity redesign |
| Repeated feedback item | Content revision |
