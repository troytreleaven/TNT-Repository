### 11 Hacks to Make OpenClaw Go From Useless to AGI
**Author:** Alex Finn (@AlexFinn)
**Date:** Saturday, February 21, 2026
**Source:** https://x.com/alexfinn/status/2025302022749389282
**Saved by:** Troy
**Status:** For Implementation

#### Key Hacks:
1. **Multi-model setup** - Use Opus for brain, different models for muscles (Codex for coding, Minimax for research, Qwen for creative)
2. **Local hosting** - Better than VPS for quick file sharing and workflows
3. **Channel strategy** - Telegram for quick, Discord for deep work
4. **Reverse prompting** - Ask "what's next?" instead of just commands
5. **Vibe coding** - Let AI use coding tools itself
6. **Custom Mission Control** - Build your own tooling
7. **Audit everything** - Run tasks through AI to find improvements
8. **Scale gradually** - Start cheap, upgrade as needed
9. **Skip email access** - Security risk, minimal value
10. **Skip X bot accounts** - API restrictions
11. **Have fun!** - Experimentation over monetization
