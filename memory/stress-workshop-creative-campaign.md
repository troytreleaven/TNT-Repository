### Daily vs Lifetime Budget Recommendation

**Recommendation: DAILY BUDGET**

**Rationale:**
- Better control over spend pacing
- Easier to adjust based on performance
- Meta's algorithm optimizes daily better for lead gen
- Less risk of overspending in high-converting periods

**When to Use LIFETIME:**
- If running time-bound promotion with fixed end date
- If you have specific events (webinar sign-ups with deadline)

---

### Ad Rotation Settings

**Recommendation:**

| Setting | Value | Rationale |
|---------|-------|------------|
| **Optimization Goal** | Conversions (Lead) | Primary goal = Workshop registrations |
| **Ad Rotation** | Rotate indefinitely | Let algorithm test and find winners |
| **Delivery Type** | Advantage+ Shopping/Advantage | Let Meta optimize placements automatically |
| **Conversion Event** | Lead | Track via Pixel or Conversions API |

**A/B Testing Strategy:**
- Test 2-3 creatives per ad set initially
- After 7 days, pause underperformers (CPA > 2x benchmark)
- Scale winners with +50% budget
- Test new creatives weekly

## 5. Quality Review (Quinn's Analysis)

### What Makes This Proven?

**1. B2B Meta Ad Best Practices (2025-2026)**
- Using Meta's native job title targeting combined with interest targeting
- Following the "creative-first" approach favored by Meta's 2026 algorithm
- 40/30/30 budget split across awareness/consideration/conversion (industry standard)
- Retargeting with warm audiences (website visitors = highest conversion intent)

**2. Emotional Resonance**
- Problem-aware messaging (leaders feeling overwhelmed)
- Social proof (500+ leaders, 94% satisfaction)
- Transformation narrative (surviving → thriving)
- Community building (peers who "got your back")

**3. Lead Generation Optimization**
- Lead ads reduce friction (no landing page needed)
- Multiple lead form variations for testing
- Low-commitment entry points (assessment, toolkit) before workshop pitch

---

### What Are the Risks?

| Risk | Likelihood | Mitigation |
|------|------------|------------|
| **High CPM for niche targeting** | Medium | Use broad targeting with creative optimization; start with interest-based, then narrow |
| **Low conversion rates** | Medium | Test multiple lead forms; optimize for quality not just quantity |
| **Ad fatigue** | High | Refresh creatives every 2-3 weeks; test new hooks |
| **Budget inefficiency** | Low | Start with $500/month, scale what works |
| **Algorithm learning phase** | High | Allow 7-14 days for optimization before major changes |
| **Competition for HR/L&D audience** | Medium | Differentiate with emotional hooks vs. generic "training" ads |

## Key Metrics to Track

| Metric | Target | When to Act |
|--------|--------|-------------|
| **CPM (Cost per 1,000 impressions)** | < $15 | If >$25, broaden targeting |
| **CTR (Click-through rate)** | > 1% | If <0.5%, test new creative |
| **CPC (Cost per click)** | < $2 | If >$3, optimize landing page |
| **CPL (Cost per lead)** | < $30 | If >$50, review audience/creative fit |
| **Workshop Registration Rate** | > 5% of leads | If <3%, improve follow-up sequence |
| **ROAS (Return on Ad Spend)** | > 3x | Track post-workshop enrollments |

## Notes

- **Meta's 2026 Algorithm:** The new Andromeda algorithm favors creative quality over narrow targeting. Invest in strong visuals and hooks.
- **Privacy:** With iOS updates, rely less on pixel data alone—prioritize first-party data (email list) for lookalikes.
- **Testing:** Always test at least 2 headlines + 2 visuals per ad set. Let data decide.
- **Follow-up:** Lead ads are just the start. Have email sequence ready to nurture leads to registration.
